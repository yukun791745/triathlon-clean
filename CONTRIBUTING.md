Thank you for considering contributing to triathlon-clean! We welcome contributions from everyone.

How to contribute
1. Fork the repository and create a branch for your change (e.g. `feature/my-feature`).
2. Make your changes and write tests where applicable.
3. Commit with clear messages. Follow conventional commits where possible (e.g. `feat:`, `fix:`, `chore:`).
4. Push your branch and open a pull request against `main`.

Code review and merging
- Please open a PR with a clear description and link to any related issue.
- At least one approval is required and the CI (`CI`) must pass before merging.

Communication
